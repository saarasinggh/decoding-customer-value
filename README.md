# Decoding Customer Value: SQL-Driven Retention Strategy
IIT Guwahati | Consulting & Analytics Club | Summer Projects '26

## Project Overview
Customer segmentation and retention strategy for a D2C fashion brand with ~3,900 US customers using Python, SQL, and Power BI.

## Key Finding
The brand's discount problem is an acquisition mix problem. Female customers show 0% promo dependency and 71.9% 
loyalty. Male customers (68% of base) show 63% promo dependency and 25.6% loyalty.

## Deliverables
- Python notebook: EDA and FE
- SQL queries: Segmentation answering 5 key business questions  
- Power BI dashboard: 4-panel founder dashboard

## Metrics Built
- ORR (Organic Revenue Rate): 57.3%
- LCE (Loyalty Conversion Efficiency): 39.2%
- SMP (Segment Migration Potential)

## Tools
Python (pandas, scipy) | MySQL Workbench | Power BI Desktop
